# NVIDIA Stock Performance Analysis (2020-2025)
## Project Overview
This project uses Python to analyze NVIDIA's stock performance against competitors (AMD) during the AI boom. It provides insights into investment returns and market volatility.
## How to Use
1. Install requirements: pip install yfinance pandas matplotlib
2. Run the Jupyter Notebook: ACC102_NVIDIA_Stock_Analysis.ipynb
## Key Findings
NVIDIA has shown significant decoupling from competitors since early 2023, driven by the generative AI surge.
